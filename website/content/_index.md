# Project Gaia

Project Gaia is a distributed smart agriculture cooperative system. 

It consists of several components:
- **Farm**: Lightweight edge software running on hardware at the farm.
- **Hub**: Central cloud management system.
- **Shared Pkg**: Shared libraries for mTLS, protocols, and syndicate logic.
